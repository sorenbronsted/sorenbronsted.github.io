Detailed Description

Class Config2 is a wrapper for php ini file functions. It uses magic get and set method to access the values from the ini file. Usage:

$config = new Config2('/path/to/config/file.ini');
$value = $config->section_name; // read a value
$config->section_name = 10; // set a value
