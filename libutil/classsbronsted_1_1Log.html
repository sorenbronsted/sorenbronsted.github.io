Detailed Description

Class Log writes information to a LogWriter. The log level are numbered so that ERROR i lowest and DEBUG is highest, and the output i written only if the given level <= the default level. It depends on the Config object in the DiContainer. Usage:

$dic = DiContainer::instance();
$dic->log = Log::createFromConfig();

Ini file configuration:

[log]
defaultLevel = error # default level is error and must be error|warn|info|debug
writer = SomeWriter # default writer i ConsoleWriter
# optional setting different loglevel on for different classes
debug[] = TestClass1,TestClass10
info[] = TestClass2
error[] = TestClass3
